Berechnung der dissipierten Arbeit je Zyklus¶

In den Vorlesungsunterlagen haben wir die dissipierte Arbeit pro Zyklus fr einen viskosen Dmpfer berechnet. Dazu haben wir das Integral

$$ W_\text{D} = \int \limits_0^T -F_\text{D} \dot{u}\, \text{d}t $$

bestimmt. Die Periodendauer $T$ teilt sich in einen Auslenkungs- und einen Rckstellungsast auf. Wir knnten also auch schreiben

$$ W_\text{D} = \int \limits_0^{T/2} -F_\text{D} \dot{u}\, \text{d}t + \int \limits_{T/2}^{T} -F_\text{D} \dot{u}\, \text{d}t $$

Die folgende Abbildung illustriert die Arbeit unter diesen beiden sten. Mit den entsprechenden Vorzeichenregeln ergibt sich als Differenz die dissipierte Arbeit als zwischen den Teilsten eingeschlossene Flche.

Schubmodul und Dämpfung bei mittleren Dehnungen¶

Handbuchformeln [Grundbau Taschenbuch] fr bindige Bden (rollige Bden bei $I_\text{P} = 0$), wobei Scherdehnungsamplitude $\gamma$ und $I_\text{P}$ in Prozent einzusetzen sind:

$$ \frac{G}{G_\text{max}} = \dfrac{1.03}{1+\dfrac{19}{1+(I_\text{P}/15)^{1.3}} \gamma^{0.8}} \leq 1 \qquad \gamma \leq 1\,\% $$$$ D = 2 + \dfrac{24.5 - I_\text{P}/5}{1 + \dfrac{1}{(7.4-I_\text{P}/10) \gamma^{0.8}}} \leq 1 \qquad \gamma \leq 1\,\%,\ I_\text{P} \leq 50\,\% $$

Als Ergebnisse folgen das Verhltnis $G/G_\text{max}$ einheitenlos und Dmpfung $D$ in Prozent.

G_Gmax = lambda GA, IP: 1.03/(1+(19*GA**0.8)/(1.0+(IP/15.0)**1.3))
D = lambda GA, IP: 2 + (24.5-(IP/5.0))/(1.0+1.0/((7.4-(IP/10.0))*GA**0.8))

Schubmodul bei sehr kleinen Dehnungen¶

Nach Grundbau-Taschenbuch

n = 0.5   # typical value (model) for sand
S = 625.0   # typical value (model) for sand
ee = 0.8   # void ratio
Ip = 0.36   # plasticity index (nondimensional)
Fe = 1/(0.3+0.7*ee**2)   # function (model)
sigma_eff0 = 1e6   # mean effective stress
p_atm = 1e5   # atmospheric pressure
GmaxNC = S*Fe*p_atm*(sigma_eff0/p_atm)**n   # normally consolidated
OCR = 5   # overconsolidation ratio
k= Ip /(1+3*Ip**2)**(0.5) # typical relation for sand
GmaxOC = GmaxNC*OCR**k   # overconsolidated

print('GmaxNC = {:2.2f} MPa,   GmaxOC = {:2.2f} MPa'.format(GmaxNC*1e-6, GmaxOC*1e-6))

GmaxNC = 264.23 MPa,   GmaxOC = 432.02 MPa

Zusatzaufgabe (für Programmierfans)¶

Simulieren Sie eine Be- und Entlastungsphase mit dem hypoplastischen Materialgesetz [W. Fellin: Hypoplasticity for beginners].

$$ \dot{\sigma}=C_1\sigma\dot{\varepsilon}+C_2\sigma|\dot{\varepsilon}| $$

Be- und entlasten Sie das Material also jeweils mit einer konstanten Dehnrate ausgehend von $\sigma_0 = 100$ kPa bis auf eine maximale Spannung von 1 MPa. Danach entlasten Sie das Material wieder auf die Ausgangsspannung. Was passiert fr dieses einfache Modell, wenn Sie weitere gleichartige Zyklen anschlieen? Ist das sinnvoll?

Verwenden Sie die Parameter $C_1 = 2 C_2 = 400$.

Musterlösung¶

Die folgende Lsung reproduziert Fig. 4 aus [Fellin2002]. Achtung: hier sind mechanische Vorzeichen gewhlt (Zug ist positiv). Hier knnen fortschrittlichere numerische Algorithmen eingesetzt werden (siehe Vorlesung Numerische Methoden). Fr das einfache Beispiel gengen allerdings einfache Verfahren.

initial_stress = -3.4e3
stress_hat = -300e3 - initial_stress
initial_strain = 0.0
#
C1 = -775
C2 = -433
#
n_steps = 1000
time = np.linspace(0,0.5,n_steps)
#
strain = np.zeros(n_steps)
stress = stress_hat * np.sin(2*np.pi*time) + initial_stress  # Hier Sinus. Da ratenunabhngig, wre auch bilineare Funktion oder Vergleichbares mglich.

strain[0] = initial_strain
for i,t in enumerate(time[0:-1]):
    dt = time[i+1] - time[i]   # Zeitschritt
    dsigma = stress[i+1] - stress[i]   # Spannungsinkrement im Zeitschritt
    mean_sigma = (stress[i+1] + stress[i])/2   # Mittlere Spannung im Zeitschritt
    CC = C1 + np.sign(dsigma)*C2   # Unterscheidung Be- und Entlastung
    de = dsigma/(mean_sigma*CC)   # Dehnungsinkrement; zeitschrittunabhngig (ratenunabhngige Plastizitt); hysteretische Dmpfung
    strain[i+1] = strain[i]+de

#HIDDEN
fig, ax = plt.subplots();
ax.plot(-stress/1e3,strain*100);
ax.set_xlabel('$-\sigma$ / kPa');
ax.set_ylabel('$\epsilon$ / %');
fig.tight_layout();
